이 문서는 Phase 2B 최종 전략의 모든 설정값과 파라미터를 기록합니다.

## 1. 최종 선정 전략: Hybrid (Vol 12%)

| 파라미터 | 값 | 설명 |
|---|---|---|
| **신호 스무딩 기간** | 40일 | 원본 신호에 적용되는 이동평균 기간 |
| **레버리지 범위** | (0.2, 2.8) | 최소/최대 레버리지 |
| **임계값** | 0.10 (10%) | 레버리지 조정 임계값 |
| **목표 변동성** | 12.0% | 포트폴리오의 목표 연간 변동성 |
| **거래비용** | 0.01% | 레버리지 변경 시 발생하는 거래비용 |
| **신호 지연** | 1일 | 신호 생성과 거래 사이의 시간 차이 |
