## 6. 배포 권장사항

### 6.1 배포 방식

1. **1단계 (테스트):** 소규모 자본으로 1개월 테스트
2. **2단계 (확대):** 성과 확인 후 자본 확대
3. **3단계 (최적화):** 실제 거래 데이터로 재최적화
4. **4단계 (운영):** 정상 운영

### 6.2 모니터링 항목

- **일일:** Sharpe, 수익률, 변동성, 최대 낙폭
- **주간:** 신호 품질, 거래비용, 레버리지
- **월간:** 성과 분석, 파라미터 재검토, 리스크 평가

### 6.3 위험 관리

- **손절매:** -5% 손절매 메커니즘
- **포지션 제한:** 최대 50% 제한
- **레버리지 제한:** 0.5x ~ 2.5x 범위
- **긴급 중단:** 일일 손실 -10% 시 중단
