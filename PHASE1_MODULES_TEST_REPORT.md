# Phase 1 모듈 종합 테스트 보고서

## 1. 개요

이 보고서는 검증된 새로운 베이스라인(Sharpe 4.24)을 기반으로 Phase 1의 주요 모듈들을 테스트한 결과를 정리합니다.

| :--- | :--- |

| **거래비용** | 0.01% (레버리지 변동 시) |

## 3. 모듈별 테스트 결과

### 3.1. Backtester 모듈

**목적:** 거래비용 모델 검증 및 Walk-Forward 최적화

| 시나리오 | Sharpe | 변화율 |
| :--- | :--- | :--- |
| No Cost | 4.5166 | +6.48% |
| Low Cost (0.01%) | 3.6631 | -13.73% |
| Medium Cost (0.05%) | 3.2712 | -22.95% |
| High Cost (0.10%) | 2.4378 | -42.54% |

**결론:** 거래비용 0.01% 수준에서 Sharpe 3.66으로 실현 가능한 성과를 보입니다.

## 6. 주요 인사이트

1. **단기 신호가 효과적:** 5일 모멘텀이 20일, 30일보다 우수
2. **변동성 조정 중요:** 변동성 기반 레버리지 조정이 일관되게 개선
3. **평균회귀는 부적합:** 이 데이터에서는 모멘텀 기반 전략이 우수
4. **거래비용 영향 큼:** 0.01% → 0.10% 변화 시 Sharpe 45% 감소
5. **다중 알파 결합:** 개별 알파보다 결합이 더 안정적
