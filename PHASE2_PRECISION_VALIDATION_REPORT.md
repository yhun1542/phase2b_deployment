### 2.1 Expanding Window 정규화

**구현 코드:**
```python
for i in range(len(data)):
    past_data = data.iloc[:i+1]  # i시점까지의 과거 데이터만
    min_val = past_data.min()
    max_val = past_data.max()
    norm_val = (data.iloc[i] - min_val) / (max_val - min_val)
```

**검증 결과:**
- ✅ **구현 정확성: 올바름**
- ✅ **룩어헤드 바이어스: 제거됨**
- 시간에 따라 정규화 범위가 증가 (예상된 동작)
